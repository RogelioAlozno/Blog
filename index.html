COMERCIO ELECTRÓNICO

Publicado el 12 de junio, 2021

¿Qué es el Comercio Electrónico?

Es toda la compra y venta que se realiza por medio del internet o la web. Estas relaciones comerciales se pueden dar entre solo organizaciones o entre organizaciones y consumidores, algunas terminologías que usualmente escucharemos referente a este tipo de negocio son: “B2B” o “B2C”.

Estas transacciones comerciales se dan mediante plataformas en línea. En dichas plataformas podemos encontrar todos los productos que se tiene disponibles para la venta, independientemente el tipo de empresa que sea, así serán los productos que ofrezcan a los consumidores.

¿Qué productos o servicios se pueden vender?

Se puede ofrecer cualquier producto o servicio, todo depende de la estrategia de marketing digitar que se utilice para poder llegar y abarcar un buen segmento del mercado.

¿Qué plataformas e-commerce son las más utilizadas?

Existen muchas plataformas que se pueden utilizar para este tipo de negocio, como es de suponer, todas tienen ventajas y desventajas. Algunas de ellas tienen una versión gratuita pero como es de esperar son limitadas las opciones que ofrece. Para poder disponer de todas las funcionalidades que ofrecen dichas plataformas se puede acceder por medio de un pago mensual o anual.

Las plataformas más utilizadas a nivel mundial son:
  • WooCommerce
  • Magento
  • Shopify
  • Wix

Todas y cada una de estas plataformas son fáciles de utilizar ya que solo es cuestión de ir editando la información que nosotros deseemos aparezca y sea visible para nuestros consumidores, a todas hay que agregarles una forma de pago. El pago se puede dar de dos formas y puede ser por medio de una pasarela de pago o sin pasarela de pago.

¿Las plataformas que nos ofrece la web será el único medio para iniciar un ecommerce?

Evidentemente no es el único medio por el cual podamos iniciar un ecommerce, existe otro medio por el cual podemos iniciarlo desde cero y es por medio de “HTML”, “CSS” Y “JavaScript”.

HTML:

Este es un lenguaje de programación que se basa en etiquetas, en las cuales introducimos la información e imágenes que deseemos. Existe una infinidad de etiquetas, algunas de ellas son: Etiquetas de títulos, listas desordenadas, imágenes, videos, enlaces, entre otras.

CSS:

Es el lenguaje que nos sirve para darle estilo a nuestro ecommerce, para darle una apariencia muy personalizada y profesional. Nos permite agregar márgenes, seccionar nuestra página, centrar las imágenes, agregar estilo, tamaño y color de la letra.

JAVASCRIPT:

Este es un lenguaje de programación que permite la interacción y el dinamismo en la página web, bien puede ser un ecommerce o un blog.

¿Qué se necesita para empezar a montar el ecommerce?

Para empezar a estructurar un ecommerce, se debe contar con los tres lenguajes antes mencionados, vamos a crear una carpeta la cual podemos nombrar con el ecommerce que se está creando, esta carpeta se puede albergar en los documentos o en el escritorio, no hay problema de su ubicación, dentro de esta carpeta se crean como mínimo cuatro carpetas las cuales las podemos nombrar “CSS”, “js”, “Index” e “imágenes”. La carpeta del Index, corresponde a la carpeta del HTML y es la carpeta principal que nos conecta directamente a internet, allí podemos ir viendo los avances que se estarán realizando. La carpeta de imágenes como su nombre lo indica, sirve para albergar todas y cada una de las imágenes que utilizaremos en la estructura.

IMÁGENES ICO:

Estas imágenes son las que se deben utilizara para ilustrar o representar como una forma de logotipo nuestro ecommerce. La abreviatura “ICO” se deriva de la palabra icono, como bien sabemos un ícono sirve para representar una carpeta en nuestro escritorio, pero en este caso lo vamos a utilizar para representar nuestro sitio web.

IMÁGENES PNG:

son las que se deben utilizar para editar nuestro ecommerce, ya que su resolución es muy alta y sobre todo el peso de estas imágenes es muy bajo, por lo tanto, es ideal ya que al momento de tener toda la estructura y se desee subir a la web, se pueda realizar sin ningún problema.

GITHUB DESKTOP

Para crear un ecommerce se necesita de los tres lenguajes antes mencionados, pero adicional a ello se necesita un espacio donde albergarlo, en este caso se puede utilizar “GitHub Desktop”, este programa sirve como un repositorio local.

GITHUB:

Permite crear repositorios de forma gratuita en la web, además permite que muchas personas en todo el mundo puedan interactuar en el proyecto que se está desarrollando.

GOOGLE ANALITIC:

Es una excelente herramienta que nos permite llevar un análisis detallado de las personas que visitarán nuestro ecommerce, nos permite saber de qué lugar visitan la página, cuántas personas por minuto la frecuentan, qué tanto tiempo pasan dentro de la página, entre otros. Son muchos datos estadísticos que nos permite llevar, dependerá de nosotros qué funciones le habilitemos.

Lo primero que debemos hacer es crear una cuenta en Google analitic, posteriormente debemos copiar un fragmento de JavaScript y debe ir antes de la etiqueta de “head”.
